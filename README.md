# docker-lamp

Development environment built with docker-compose

## setup

### Clone this repository

```
git clone https://github.com/nircat/docker-lamp.git
cd docker-lamp
docker-compose up
```

### It's done!

Go to http://localhost:3000/ and you can see PHP works.

## Version
